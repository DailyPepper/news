## Запуск

Для начала нужно склонировать репозиторий:

```
git clone https://github.com/DailyPepper/news
```

Перейти в деректорию new:

```
cd new
```

После выполнить команду в терминале:

```
yarn install
```

Затем выполнить команду для запуска:

```
yarn start
```
